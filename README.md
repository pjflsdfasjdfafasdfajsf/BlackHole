Simple black hole rendering in C. Powered by [Simple DirectMedia Layer](https://www.libsdl.org/) by Sam Lantinga and co.

Other than a C compiler and Make you need to have [SDL3](https://github.com/libsdl-org/SDL), [SDL3_ttf](https://github.com/libsdl-org/SDL_ttf) and [SDL3_shadercross](https://github.com/libsdl-org/SDL_shadercross) installed on your system.
```
make 
./BlackHole
```
